# homebrew-tap

Homebrew tap for [OnekoMac](https://hellov3an.github.io/OnekoMac/) — the pixel art cat that lives in your menu bar.

## Install

```sh
brew tap hellov3an/tap
brew install --cask onekomac
```

Or in one line:

```sh
brew install --cask hellov3an/tap/onekomac
```

## Update

```sh
brew upgrade --cask onekomac
```

## Uninstall

```sh
brew uninstall --cask onekomac
```
